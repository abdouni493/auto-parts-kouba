2. Create test worker account

4. Verify store is locked

## 💡 Tips & Tricks

### Quick Reference
```
Supabase URL: https://app.supabase.com
Project: zpbgthdmzgelzilipunw
Critical: Email confirmation OFF
Worker role: 'employee'
Admin role: 'admin'
```

### Browser Console Commands
```javascript
// Check stored credentials
localStorage.getItem('worker_test@example.com')

// Clear all data
localStorage.clear()

// Check auth token
localStorage.getItem('sb-access-token')
```

### Keyboard Shortcuts
- F12: Open DevTools
- Ctrl+Shift+J: Console
- Ctrl+Shift+K: Console (Firefox)
- Cmd+Opt+J: Console (Mac)
